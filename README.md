# Academic Paper Website Template

Made by [Lavínia Rocha Lima](https://github.com/lavinia-rl), Computer Engineering student in Federal University of Goiás 🎀


## Introduction

A clean, responsive website template for academic papers, ready for GitHub Pages.

This template provides a professional layout for presenting academic research online, including title, authors, abstract, motivation, evaluation, related work, and media.

## Features

- Academic layout
- Affiliation and contribution markers
- Buttons for PDF, arXiv, YouTube, GitHub, Hugging Face
- Media support (YouTube + images)
- Related work section
- Responsive + GitHub Pages ready

## How to Customize

### Content

Open `index.html` and you'll see the paper title, author names with their links and affiliation markers, just update those with your own details. The abstract, motivation, evaluation, and related work sections are filled with placeholder text, so feel free to replace it with your actual content. There's a YouTube video already embedded, simply swap the video ID in the link to show your own. If you want to add images, just drop them in using the image class. The buttons for PDF, arXiv, GitHub, and others are ready for you to update the links and icons. The related work section is set up with examples, edit the project titles, descriptions, and author names as needed. The page title, favicon, and meta description live in the `<head>` section. And please keep the footer attribution intact, it helps others discover the template too!

### Styling

Edit `style.css` file to change colors, fonts and layout settings.

## How to Preview

**VS Code**: Install Live Server extension → right-click on `index.html` → Open with Live Server

**Python**: `python -m http.server 8000` → open `http://localhost:8000`

## How to Deploy (GitHub Pages)

1. Create a repository on GitHub
2. Upload the files of this repository
3. Go to Settings → Pages → select `main` branch → Save
4. Your site will be at `https://your-username.github.io/repo-name/`
